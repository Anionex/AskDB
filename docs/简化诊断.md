# 浏览器 Console 正确诊断命令

## 方法1: 通过 React DevTools

### 1. 安装 React DevTools（如果没有）
在 Chrome/Edge 扩展商店搜索 "React Developer Tools" 并安装

### 2. 打开 Components 标签
F12 → Components (React DevTools)

### 3. 选择任意组件，在 Console 中执行：
```javascript
$r.props
```

---

## 方法2: 直接检查页面元素

在 Console 中执行：

```javascript
// 方法1：检查是否有工具调用的 DOM 元素
document.querySelectorAll('[class*="ant-collapse"]').length
// 如果返回 > 0，说明组件渲染了

// 方法2：查找工具调用文本
document.body.innerText.includes('调用了')
// 如果返回 true，说明文本存在

// 方法3：查找所有可能的工具调用容器
Array.from(document.querySelectorAll('div')).filter(d => d.innerText.includes('工具'))
```

## 方法3: 查看 Network（最可靠）

1. **打开 F12 → Network 标签**

2. **清空日志** (点击 🚫 图标)

3. **发送查询**: "列出所有数据库表"

4. **找到 `stream` 请求** (名称类似 `stream?message=...`)

5. **点击该请求 → Response 标签**

6. **复制前 10 行内容给我！**

特别是找这几行：
```
data: {"type":"tool_call_start",...}
data: {"type":"tool_call_result",...}
```

1. **打开 F12 → Network**
